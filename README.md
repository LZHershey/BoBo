# BOBO: Movie Recommendation and Party Starter

BoBo is a social media app for cinephiles. Users can invite friends to parties, which will prompt them to vote on movies they want to watch with BOBO providing the outcome. These movies will be recommended based on watching preferences, as well as streaming services available to those in the party. Finally, an easy answer to â€œwhat should we watch tonight?"

## Run Locally

Clone the project

```bash
  git clone git@github.com:Gonzos-Greenery/BoBo.git
```

Go to the project directory

```bash
  cd BoBo
```

Install dependencies

```bash
  yarn install
```

Start the app (Press i to open an ios simulator if you have XCode)

```bash
  expo start 
```
